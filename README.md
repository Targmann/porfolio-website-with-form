# porfolio-website-with-form
This project was coded in HTML, CSS, and JavaScript from scratch. It demonstrates creative design skills and complex CSS features including grid positioning, flexbox styling, relative positioning, and media queries that allow the website to respond to different screen sizes for full mobile display and functionality, as can be demonstrated in the Developer Tools. For privacy concerns, personal contact information has been replaced with fake example text. At the top of the page is a button that is used to toggle between two different designs. Comments in the CSS code describe implemented features and improvements from previous versions.
